Platform Guide / Admins

Admin Journey

Admins orchestrate tournament operations: event setup, team acceptance, match lifecycle, and official results.

Interaction Flow

1. Tournament creation

Admin creates tournaments with game, format, slots, region, and timeline metadata.

2. Registration moderation

Admin approves or rejects captain submissions and finalizes the participant field.

3. Match creation

Admin creates fixtures with round labels, teams, schedule windows, and map/mode context.

4. Results upload

Admin submits scores or winner data; disputes can be moderated before final locking.

5. Publish + monitor

Standings, bracket progression, and schedule updates propagate to public and participant pages.

System Handling

  • - Admin endpoints are role-gated and expose management capabilities not available to players.
  • - Match controllers validate result payloads before passing them to game-specific resolvers.
  • - Progression services update downstream structures like brackets and next-round fixtures.
