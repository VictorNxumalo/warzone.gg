Platform Guide / Players Without Teams

Player Journey (No Team Yet)

A signed-in user without team membership can manage a player identity and move toward team participation.

Interaction Flow

1. Register + sign in

User creates account, authenticates, and gets a role-aware navigation state.

2. Player profile and eligibility

User can view player profile and account details while not tied to a squad.

3. Join path

User discovers teams, receives invites, or sends join requests depending on team governance rules.

System Handling

  • - Session persistence stores token and user object to keep sign-in state across pages.
  • - Team ownership checks drive visibility of team-specific actions and pages.
  • - Invite/request controllers and roster governance services enforce membership transitions.
